# 04 — Tc と TTT 閾値理論

## 1. TTT の閾値条件

TTT における安定条件：
```
|Δv| < v_c  →  系は安定（超電導状態を維持）
|Δv| ≥ v_c  →  系は崩壊（超電導消失）
```

ここで Δv = 外乱による各点のベクトル変位。

---

## 2. Tc の TTT 的定義

超電導転移温度 Tc は：

```
Tc = 閾値温度
  = 熱的外乱（波の振幅）が Cooper 対（00 状態）を
    破壊し始める温度
```

TTT の「時間 = 波の振幅」との対応：
```
k_B · Tc ≈ 2Δ（ペアギャップ）

TTT: 熱振動の振幅 ∝ k_B T が
     「00 到達を妨げる閾値振幅 v_c」に等しくなる温度 = Tc
```

---

## 3. 構造パラメータと Tc の TTT モデル

### 3.1 ベクトル均衡度（TTT-EQ）

正八面体の均衡度を定義する：

```
EQ = 1 − D/d₀

EQ = 1.0  →  完全均衡（Tc 最大を期待）
EQ = 0.0  →  完全崩壊（超電導なし）
```

### 3.2 Tc の TTT 予測式（仮説）

```
Tc(TTT) = Tc_max · EQ · f(t) · g(δ)

ここで：
  Tc_max  = 材料クラスの最大 Tc（経験値）
  EQ      = TTT ベクトル均衡度（0〜1）
  f(t)    = tolerance factor 補正：f(t) = exp(−β|t−1|)
  g(δ)    = 酸素欠損補正：g(δ) = (1−δ/δ_max)^γ
```

## 4. 主要ペロブスカイト超電導体への適用

### 4.1 YBCO 系（YBa₂Cu₃O₇₋δ）

| 酸素欠損 δ | Tc (実測) | 構造 | TTT 的状態 |
|---|---|---|---|
| 0.0 | 92 K | 斜方晶（Orthorhombic） | ほぼ均衡、わずかな非等価 |
| 0.2 | 60 K | 斜方晶 | 均衡度低下 |
| 0.5 | 0 K | 正方晶（Tetragonal） | 正方→TTT軸等価化→ペア破壊 |

注目点：δ = 0 の斜方晶が Tc 最高。TTT 的には「微妙な x-y 非等価」が  
d 波ペア形成に最適なベクトル構成を与えている。

### 4.2 LSCO 系（La₁₋ₓSrₓCuO₄）

| Sr 量 x | Tc (実測) | TTT 的解釈 |
|---|---|---|
| 0 | 0 K | 反強磁性：スピンベクトル局所整列、全体ゼロに非収束 |
| 0.05 | ~10 K | 局所的「00」が形成され始める |
| 0.15 | 38 K | ベクトル系が「00 到達」に最も近い最適状態 |
| 0.25 | ~15 K | 余剰キャリアが均衡を過剰に撹乱 |
| 0.30 | 0 K | 金属状態、Cooper 対不形成 |

### 4.3 BaBiO₃ 系（Ba₁₋ₓKₓBiO₃）

特徴：電荷不均化（CDW）との競合。

```
Bi³⁺ と Bi⁵⁺ が交互配列（x=0）：
  ベクトル系が「正の Bi」と「負の Bi」に二極化
  → TTT「00」の空間的分離状態
  → 超電導消失

K ドープ（x≈0.4）で CDW 抑制：
  二極化が解消されベクトル均衡に向かう
  → Tc ≈ 30 K の超電導発現
```

## 5. TTT 閾値と臨界磁場の関係

上部臨界磁場 Hc₂ も TTT で記述できる：

```
Hc₂ = Φ₀ / (2π · ξ²)

TTT 的：磁場 H が与える「ベクトル変位」が閾値を超えると
        9点系が崩壊（超電導消失）

Hc₂ ∝ 1/ξ²  ∝  1/(Cooper 対サイズ)²
    = 「00 状態の空間的凝縮度」の 2 乗に比例
```

高 Tc 材料（小さい ξ）= より強い磁場まで「00 状態」を保持。

## 6. TTT 閾値理論の予測と検証可能性

### 予測1：tolerance factor と Tc の相関

```
Tc が最大となる t の値は 1.0 よりわずかに小さい
（正確な OOπ よりも「正四面体への収縮途中」が最適）
```

これは実際の銅酸化物 t ≈ 0.88〜0.92 に対応。

### 予測2：最適歪みの存在

完全正八面体（t=1）よりも軽微な傾き歪みがある方が Tc が高い。  
→ **「ベクトル均衡からの適度な逸脱」が超電導を促進する**。  
実験的にも確認されている（YBa₂Cu₃O₇ の斜方晶変形）。

### 予測3：TTT-D と Tc の逆相関

```
Tc ↑  ←→  TTT-D ↓（ベクトル歪み指数が小さい）
```
