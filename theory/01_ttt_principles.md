# 01 — TTT 基本原理とペロブスカイトへの適用

## 1. TTT の核心：ベクトル総和ゼロ

トリテトラ理論（TTT）の根本命題：

```
あらゆる空間・時間・構造・生命・社会における
ベクトルの総和は 0 である。
```

三軸表現：
```
R + I + J = 0
```

ここで R = 実数軸、I = 第一虚数軸、J = 第二虚数軸。

### OOπ（球面への完全展開）

```
∯ v dΩ = 0
```

全立体角にわたるベクトル積分がゼロのとき、空間は球面として完全展開する。これを **OOπ** と定義する。

---

## 2. ペロブスカイト構造とのマッピング

### ABO₃ の基本構造

ペロブスカイト（ABO₃）は以下で構成される：
- **A サイト**：大きなカチオン（Ba, La, Ca, Sr...）
- **B サイト**：遷移金属イオン（Cu, Ti, Mn, Fe...）
- **O サイト**：酸素、B を中心とした正八面体を形成

### TTT 変数との対応

| TTT 変数 | ペロブスカイト構造要素 | 物理的意味 |
|---|---|---|
| R 軸（実数軸） | Cu-O-Cu 結合の x 方向 | 電荷移動積分 t_x |
| I 軸（第一虚数軸） | Cu-O-Cu 結合の y 方向 | 電荷移動積分 t_y |
| J 軸（第二虚数軸） | 頂点酸素・c 軸方向 | 層間結合 t_z |
| 双極のゼロ（00） | B サイトイオン（反転対称中心） | Cooper 対の凝縮点 |
| ベクトル総和 = 0 | 完全立方対称 Pm3̄m | 最高 Tc 状態 |

## 3. tolerance factor の TTT 的解釈

ゴールドシュミットの tolerance factor：

```
t = (r_A + r_O) / (√2 · (r_B + r_O))
```

TTT との対応：

| tolerance factor | 幾何学的状態 | TTT 的記述 |
|---|---|---|
| t = 1 | 完全立方（Pm3̄m） | OOπ：ベクトル完全均衡・球面展開 |
| t < 1 | 正八面体傾き（GdFeO₃ 型） | 正四面体的収縮：均衡からの逸脱 |
| t > 1 | 六方晶への転移 | ベクトル過剰：別の均衡点への収束 |

**重要な示唆**：t = 1 からの逸脱の大きさが、TTT のベクトル総和からのズレ（`|Δv|`）に対応し、Tc の低下と相関する。

---

## 4. 超電導のドーピング依存性

La₁₋ₓSrₓCuO₄ を例にとると：

| ドープ量 x | 状態 | TTT 的解釈 |
|---|---|---|
| x = 0 | 反強磁性絶縁体 | スピンベクトルが局所化・整列（局所均衡のみ） |
| x ≈ 0.05 | 絶縁体-超電導境界 | 全体のベクトル均衡が崩れ始める |
| x ≈ 0.15 | 最適ドープ（最大 Tc） | ベクトル系が「00 到達点」に最も近づく |
| x > 0.25 | 過剰ドープ（Tc 低下） | 余剰キャリアがベクトル系を撹乱 |
