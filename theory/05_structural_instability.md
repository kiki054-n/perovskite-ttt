# 05 — ヤーン・テラー効果と「点の歪み」

## 1. TTT における「点の歪み」

TTT では：
> 各点は「ベール」——格子状の場——に囲まれており最小距離を保つ。  
> 外乱により点が変位し、閾値を超えると系が破壊される。

これはペロブスカイトの **ヤーン・テラー（JT）効果** と同型。

---

## 2. ヤーン・テラー効果の TTT 的解釈

### 2.1 静的 JT 効果（点の固定変位）

遷移金属 B の電子配置が正八面体対称性を不安定化させる：

```
典型例：Mn³⁺（d⁴, 高スピン）

正八面体（対称）：
  d_{z²} と d_{x²-y²} が縮退（等価）

TTT 的：J 軸（z 方向）と R, I 軸（x, y 方向）が等価
  → OOπ に近い状態

JT 歪み後（z 伸長）：
  d_{z²} が安定化、d_{x²-y²} が不安定化

TTT 的：J 軸のみ伸長 → 等価性の破れ
  TTT-D = |Δv| > 0
```

### 2.2 動的 JT 効果（点の振動的変位）

低温では歪みが固定（静的）、高温では方向がゆらぐ（動的）：

```
高温（T > T_JT）：動的 JT
  → 点が閾値未満で振動
  → 平均的にはベクトル総和 ≈ 0
  → 超電導と競合しない

低温（T < T_JT）：静的 JT
  → 点が一方向に固定変位
  → |Δv| が有限に固定される
  → 超電導の Tc を低下させる
```

## 3. 主要材料における JT 効果と超電導の競合

| 材料 | JT 活性イオン | JT と超電導の関係 |
|---|---|---|
| LaMnO₃ | Mn³⁺ | 強い静的 JT → 超電導なし（コロッサル磁気抵抗） |
| La₁₋ₓSrₓMnO₃ | Mn³⁺/Mn⁴⁺ | x 増加で JT 抑制 → 金属転移（超電導には至らず） |
| YBa₂Cu₃O₇ | Cu²⁺ | 弱い JT（d⁹）→ d 波超電導と共存 |
| La₁₋ₓSrₓCuO₄ | Cu²⁺ | 適度な JT → 最適ドープで超電導 |
| BaBiO₃ | Bi³⁺/Bi⁵⁺ | 「呼吸モード」JT → CDW、K ドープで超電導 |

## 4. JT 歪みの TTT 定量化

### 4.1 JT 歪みパラメータ Q と TTT-D の関係

軸方向 JT 歪みパラメータ：
```
Q_JT = (2d_z − d_x − d_y) / √6
```

TTT 歪み指数との関係：
```
TTT-D = |Δv| ≈ √(2/3) · |Q_JT| · (3/d₀)

→ Q_JT が大きいほど TTT-D が大きく、Tc が低下
```

### 4.2 コード例

```python
import numpy as np

def jt_distortion(d_x, d_y, d_z):
    """
    ヤーン・テラー歪みパラメータ Q を計算
    
    Parameters
    ----------
    d_x, d_y, d_z : float
        B-O 結合長（x, y, z 方向の平均）
    
    Returns
    -------
    Q_JT : float
        JT 歪みパラメータ
    TTT_D : float
        TTT 歪み指数
    """
    Q_JT = (2*d_z - d_x - d_y) / np.sqrt(6)
    d0 = (d_x + d_y + d_z) / 3
    TTT_D = np.sqrt(2/3) * abs(Q_JT) * (3/d0)
    return Q_JT, TTT_D

# 例：LaMnO₃（強い JT 歪み）
Q, D = jt_distortion(d_x=1.96, d_y=1.96, d_z=2.28)
print(f"LaMnO₃: Q_JT = {Q:.4f} Å, TTT-D = {D:.4f}")

# 例：YBa₂Cu₃O₇ CuO₂ 面（弱い JT 歪み）
Q, D = jt_distortion(d_x=1.94, d_y=1.94, d_z=2.30)
print(f"YBCO CuO₂: Q_JT = {Q:.4f} Å, TTT-D = {D:.4f}")
```

## 5. 点の切断理論と超電導消失

TTT の「点の切断理論」：
> 「点を切る → 命が消える（死）」

超電導における対応：

| TTT | 超電導物理 |
|---|---|
| 点を切る | Cooper 対を破壊する（pair-breaking） |
| 点の変位が v_c 超過 | 外乱エネルギーが 2Δ を超える |
| 命が消える | 超電導状態の消失（抵抗の回復） |
| 偶数点の対称分割 | Cooper 対（偶数電子の対形成） |
| 奇数点の分割不能 | 孤立電子（準粒子励起）の形成 |

## 6. 反強磁性とフラストレーション

TTT の「3対4のダイナミクス」：

```
t=0：反強磁性絶縁体（3 対 4 のスピン配列的不均衡）
t=1：ドーピングで Cooper 対形成（4が2と2に分裂→均衡）
t=∞：最適超電導（総和ゼロへの収束）
```

これは LSCO のドーピング相図に対応する。

## まとめ：TTT で見るペロブスカイト超電導の統一像

```
00（双極のゼロ）
  ↓
波（格子振動・フォノン）
  ↓  ← JT 歪みはここで「点の変位」として現れる
正四面体（sp³ 局所秩序・JT 歪み）
  ↓
BO₆ 正八面体 = TTT 9点系（ペロブスカイトの基本単位）
  ↓  ← tolerance factor t が OOπ への近さを決める
OOπ（完全立方対称・超電導凝縮状態）
  ↓
Cooper 対 = 双極のゼロ（00）の量子的実現
  ↓
超電導：ベクトル総和ゼロへの巨視的収束
```
