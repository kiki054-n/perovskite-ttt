# 03 — Cooper 対 = 双極のゼロ（00）の量子的実現

## 1. TTT の双極のゼロ（00）

TTT の根源的状態「双極のゼロ（00）」：

```
+∞ + (−∞) = 0   ← 宇宙の初期状態
```

プラスとマイナス、存在と非存在が **同時に共存する** 根源的状態。

---

## 2. Cooper 対の構造

BCS 理論（1957）における Cooper 対：

```
|ψ_Cooper⟩ = Σ_k g(k) c†_{k↑} c†_{-k↓} |0⟩
```

キーとなる対称性：

| 量子数 | 電子 A | 電子 B | 対の総和 |
|---|---|---|---|
| 運動量 | +ℏk | −ℏk | **0** |
| スピン | ↑（+ℏ/2） | ↓（−ℏ/2） | **0** |
| エネルギー | ε_k | ε_{-k} = ε_k | 2ε_k（フェルミ面上） |

## 3. Cooper 対 = TTT「00」の同一性

### 対称性の完全対応

| TTT「双極のゼロ（00）」 | Cooper 対 |
|---|---|
| + と − が同時に共存 | ↑ と ↓ スピンが対 |
| +∞ + (−∞) = 0 | +k + (−k) = 0（運動量） |
| 双極性を持つ「点」 | Cooper 対（ボゾン的粒子） |
| ベールに囲まれた最小距離 | コヒーレンス長 ξ（対のサイズ） |

### 超電導転移の TTT 的記述

```
通常金属（T > Tc）：
  各電子が独立にベクトルを持つ → 総和 ≠ 0（局所的）

超電導状態（T < Tc）：
  全電子ペアが「00 状態」に凝縮 → 巨視的ベクトル総和 → 0

転移点 Tc：
  熱的外乱（波の振幅）が「00」到達を妨げる閾値温度
```

TTT の言葉：
> **「原点は出発点ではなく、到達点である」**  
> 超電導状態 = 全電子ペアが原点（00）に到達した巨視的量子状態

## 4. ペロブスカイト超電導での具体例

### YBCO（YBa₂Cu₃O₇）

```
CuO₂ 面の Cooper 対形成：

Cu²⁺ の d_{x²-y²} 軌道（TTT: OOπ の 2D 射影）
↓
ホール（+k, ↑） と ホール（−k, ↓） のペア形成
↓
d 波対称性：Δ_k = Δ₀(cos k_x - cos k_y)
  →  TTT: x と y の非等価性（I ≠ J）に対応
```

### d 波ギャップと TTT の非等価軸

BCS の s 波（等方的）ではなく d 波（異方的）である理由：

```
TTT の三軸：
  R（実数軸）= x 方向  →  cos k_x の正の寄与
  I（第一虚数軸）= y 方向  →  cos k_y の負の寄与
  J（第二虚数軸）= z 方向  →  弱い層間結合

R と I が等価（R = I）のとき s 波ギャップ（等方的）。
R ≠ I（x-y の非等価性）のとき d 波ギャップ。

銅酸化物では CuO₂ 面の x, y 方向の電子構造が
酸素との結合を通じて微妙に非等価 → d 波が安定。
```

## 5. ボゾン凝縮と TTT の「到達点」

Cooper 対はスピン 0 のボゾンとして振る舞い、  
ボース・アインシュタイン凝縮（BEC）類似の状態を形成する：

```
BEC 的凝縮：
  全 Cooper 対が単一の巨視的量子状態に落ち込む
        ↓
  TTT「原点（00）への収束」の量子的実現
```

| BEC/BCS 概念 | TTT 概念 |
|---|---|
| 凝縮体（condensate） | 00 への到達状態 |
| 秩序パラメータ ψ | ベクトル総和ゼロの度合い |
| コヒーレンス長 ξ | 「ベール」の影響範囲 |
| ペアギャップ 2Δ | 閾値エネルギー v_c に対応 |

## 6. BCS-BEC クロスオーバーと TTT

弱結合（BCS）から強結合（BEC）へのクロスオーバーは、  
TTT の「双極のゼロへの収束過程」の連続変化として理解できる：

```
BCS 極限：ξ >> a（格子定数）
  → 多数の格子点にまたがる弱い「00」状態

BEC 極限：ξ ~ a
  → 格子スケールに収縮した強い「00」状態（プレフォームドペア）

最適超電導：中間の ξ
  → 「00」到達の効率が最大
```
