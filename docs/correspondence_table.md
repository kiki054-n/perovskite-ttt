# TTT ↔ ペロブスカイト超電導 詳細対応表

## 完全対応マップ

### レベル 1：幾何学的対応

| TTT 概念 | 定式化 | ペロブスカイト対応 | 物理量 |
|---|---|---|---|
| 双極のゼロ（00） | `+∞ + (−∞) = 0` | B サイト金属イオン（反転中心） | 対称点 (0,0,0) |
| 三軸ベクトル和 | `R + I + J = 0` | Cu-O 結合の x, y, z 成分和 | 電荷移動積分 t_x+t_y+t_z |
| 正四面体第四頂点 | `V = −(R+I+J)` | A サイトイオン配置 | マデルングポテンシャル |
| OOπ（球面展開） | `∯v dΩ = 0` | 完全立方対称 Pm3̄m | t = 1 の理想ペロブスカイト |
| 正四面体角 | `cos θ = −⅓, θ = 109.47°` | B-O-B 極限歪み角 | tilting angle |

### レベル 2：電子的対応

| TTT 概念 | ペロブスカイト対応 | 超電導への影響 |
|---|---|---|
| 双極のゼロ（00） | Cooper 対（+k↑, −k↓） | 超電導秩序パラメータの起源 |
| ベクトル総和ゼロへの収束 | BCS/BEC 凝縮 | 超電導転移 |
| 原点は到達点 | フェルミ面 → 凝縮体 | 超電導状態が「到達点」 |
| 閾値 v_c | ペアギャップ 2Δ | pair-breaking エネルギー |
| 閾値超過 | T > Tc（熱的脱対） | 超電導消失 |

### レベル 3：熱的対応

| TTT 概念 | 熱的現象 | 数式的対応 |
|---|---|---|
| 時間 = 波の振幅 | 熱振動（フォノン） | k_BT ~ ℏω_Debye |
| 波の振幅が閾値以下 | T < Tc（安定） | k_BT < 2Δ |
| 波の振幅が閾値超過 | T > Tc（崩壊） | k_BT > 2Δ |
| TTT-D（歪み指数） | tolerance factor | t = (r_A+r_O)/(√2(r_B+r_O)) |

## 材料設計指針（TTT 理論からの提言）

### 高 Tc ペロブスカイトの TTT 的設計原則

1. **OOπ への近接**：tolerance factor を 1.0 に近づける（t ≈ 0.90〜0.95）
2. **9点均衡の最適化**：BO₆ 正八面体の歪み（TTT-D）を最小化
3. **双極のゼロへの誘導**：最適ドーピングで Cooper 対形成を最大化
4. **閾値の引き上げ**：ペアギャップ 2Δ を大きくするため B-O 結合の共有性を高める
5. **JT 効果の制御**：JT 活性イオンを避けるか、動的 JT を利用する

### 具体的な材料探索方向

```
高 Tc の TTT 的条件：
  EQ（均衡度）→ 最大化
  t（tolerance factor）→ 1 に近づける
  Q_JT（JT 歪み）→ 最小化
  n（キャリア密度）→ 最適化（多すぎず少なすぎず）
  dim（次元性）→ 2D（CuO₂ 面）が TTT の I, J 軸非等価性を最大化
```
